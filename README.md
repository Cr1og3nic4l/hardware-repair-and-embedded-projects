# Hardware Repair & Embedded Systems Portfolio

Hands-on hardware engineering and embedded systems projects focused on board-level diagnostics, embedded computing infrastructure, and electronic repair.

This repository documents real-world technical work including Raspberry Pi systems, AI infrastructure, and hardware-level smartphone repairs.

---

## Projects

### Raspberry Pi 5 – Portable Mini Tablet
Custom portable computing device built using a Raspberry Pi 5 integrated with a 3.5" GPIO touchscreen display inside a PLA+ enclosure.

Main aspects:
- Embedded system assembly
- GPIO display configuration
- Linux system troubleshooting
- Portable diagnostic workstation design

Project folder:
`raspberry-pi-mini-tablet`

---

### Raspberry Pi 5 – AI Discord Bot Server
Low-power dedicated infrastructure node running Python-based Discord moderation and AI bots 24/7.

Main aspects:
- Python bot deployment
- Remote server management (SSH)
- automated startup services
- lightweight infrastructure design

Project folder:
`raspberry-pi-discord-bot-server`

---

### Samsung S7 Edge – RF Chip Hardware Repair
Board-level repair involving replacement of the Qualcomm WTR4905 RF transceiver responsible for cellular connectivity.

Main aspects:
- RF subsystem diagnostics
- BGA chip replacement
- smartphone logic board repair
- hardware schematic analysis

Project folder:
`samsung-s7-rf-chip-repair`

---

### Seagate HDD – PCB Component Repair
Diagnosis and repair of a failed hard drive PCB caused by a damaged current sensing resistor (1R2).

Main aspects:
- PCB diagnostics
- SMD component replacement
- data recovery hardware repair

Project folder:
`seagate-hdd-pcb-repair`
